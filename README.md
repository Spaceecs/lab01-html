**Дисципліна:** Створення Web-сайтів (Front-end) / Web-програмування  
**Виконав:** Боцяновський Олександр

## Опис виконаної роботи

У межах практичної роботи було створено спільне CSS-оформлення для мінісайту з трьох сторінок, розробленого під час ЛР №1. Всі стилі повністю винесені у зовнішні файли, а структура коду логічно розділена на базове оформлення та адаптивні медіа-запити.

### Що було реалізовано:

1. **Глобальні стилі та дизайн-система:**
   - Створено палітру кольорів за допомогою CSS-змінних (`:root`).
   - Встановлено правило `box-sizing: border-box` для всіх елементів сайту.
   - Налаштовано єдину типографіку, скинуто стандартні зовнішні відступи (`margin: 0`) для `body`.
   - Створено клас `.container` для гнучкого центрування контенту.
2. **Макет та модулі Flexbox:**
   - **Перший приклад:** Горизонтальне меню навігації та шапка сайту (`.header-container`).
   - **Другий приклад:** Двоколонкова гнучка композиція для медіа-блоків (відео та аудіо) на сторінці "Про мене" (`.media-flex-container`).
3. **Стилізація контенту:**
   - Створено загальний стиль карток розробника (`.card`) з тінню та заокругленням.
   - Оформлено таблицю: налаштовано відступи в комірках, додано контрастний фон для заголовків та реалізовано "зебру" (чергування кольору рядків за допомогою псевдокласу `:nth-child(even)`).
   - Оформлено списки, цитати та медіа-елементи. Скрипти та відео отримали адаптивну ширину.
4. **Стилізація форми зв'язку:**
   - Поля форми отримали сучасний вигляд, однакову ширину та внутрішні відступи.
   - Реалізовано інтерактивний стан `:focus` для полів вводу (зміна кольору рамки та додавання ефекту світіння `box-shadow`).
   - Налаштовано стани `:hover` та `:active` для кнопок із використанням плавних переходів (`transition`).
5. **Адаптивність (responsive.css):**
   - Реалізовано медіа-запит `@media (max-width: 768px)` для коректного відображення сайту на планшетах та смартфонах.
   - На мобільних екранах елементи навігації та медіа-картки автоматично перебудовуються у вертикальні стеки.
   - Перевірено відсутність горизонтального скролу на мобільних пристроях.

## Інструкція із запуску проекту

Код виконується безпосередньо в браузері і не потребує встановлення додаткових серверних середовищ.

### Варіант 1. Через VS Code (З автоматичним оновленням)

1. Переконайтеся, що в редакторі встановлено розширення **Live Server**.
2. Відкрийте папку проекту `lab02-css` у VS Code.
3. Клацніть правою кнопкою миші на файл `index.html` та виберіть пункт **"Open with Live Server"**.

### Варіант 2. Звичайне відкриття файлу

1. Перейдіть у папку з проектом через провідник вашої операційної системи.
2. Двічі клацніть на файл `index.html`, щоб відкрити його у браузері за замовчуванням.
