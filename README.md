# Практична робота №1: Основи HTML

**Дисципліна:** Створення Web-сайтів (Front-end) / Web-програмування  
**Виконав:** Боцяновський Олександр

## Опис реалізованого проекту

В ході виконання практичної роботи було розроблено міні-сайт із трьох сторінок зі спільною навігацією за такою структурою:

1. **Головна сторінка (index.html):**
   - Створено базовий каркас HTML5 сторінки (усі необхідні мета-теги, визначено мову сторінки uk).
   - Використано семантичні блоки header, nav, main, footer, а також елементи figure та figcaption для оформлення зображення.
   - Додано марковані та нумеровані списки, застосовано акцентування тексту за допомогою тегів strong та em.
   - Створено таблицю "Розклад навчання" з використанням елементів та структури thead, tbody, th, td.
2. **Про мене (pages/about.html):**
   - Інтегровано мультимедійні елементи video та audio з підтримкою декількох джерел файлів (source) для кросбраузерності та додано резервний текст (fallback) для застарілих браузерів.
3. **Контакти (pages/contact.html):**
   - Створено форму зворотного зв'язку з елементами: текстове поле, email, телефон із перевіркою за шаблоном, випадаючий список select, група радіокнопок, текстова область textarea та обов'язковий чекбокс згоди з правилами.
   - Налаштовано базову HTML5 валідацію полів (required, minlength, pattern).
4. **Додаткове завдання (pages/app.js):**
   - Підключено зовнішній JavaScript файл, який додає інтерактивність кнопці відправки форми (зміна тексту на 1 секунду після натискання).

## Інструкція із запуску проекту

Увесь код виконується безпосередньо у браузері користувача. Доступні два варіанти запуску проекту:

### Варіант 1. Через VS Code (З автоматичним оновленням сторінки)

1. Переконайтеся, що у вашому редакторі встановлено розширення Live Server.
2. Відкрийте кореневу папку проекту в VS Code.
3. Клацніть правою кнопкою миші на файл index.html у провіднику та виберіть пункт "Open with Live Server".

### Варіант 2. Звичайне відкриття файлу

1. Перейдіть у папку з проектом через файловий менеджер вашої операційної системи.
2. Двічі клацніть на файл index.html, щоб відкрити його у браузері за замовчуванням.
